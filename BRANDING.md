## Typographie

### Polices Recommandées
- **Titres** : Inter Bold (sans-serif, moderne et épuré)
- **Corps de texte** : Inter Regular (lisibilité optimale)
- **Accents** : Poppins Bold (pour les slogans humoristiques)

### Hiérarchie Typographique
- **H1** : 48px, Bold, Noir
- **H2** : 36px, Bold, Noir
- **H3** : 28px, Bold, Noir
- **Body** : 16px, Regular, Gris Foncé
- **Small** : 14px, Regular, Gris Moyen
- **Slogan** : 24px, Bold, Orange Électrique

## Styles & Éléments

### Boutons
- **Style** : Arrondis (border-radius: 8px)
- **Primaire** : Fond Orange, texte Blanc
- **Secondaire** : Fond transparent, bordure Orange, texte Orange
- **Désactivé** : Fond Gris Clair, texte Gris Moyen

### Cartes Produits
- **Fond** : Blanc avec bordure grise subtile
- **Ombre** : Subtile (0 2px 8px rgba(0,0,0,0.1))
- **Hover** : Ombre augmentée, léger décalage vers le haut

### Séparateurs
- **Couleur** : Gris Moyen (#CCCCCC)
- **Épaisseur** : 1px
- **Espacement** : 24px avant/après

## Ton & Voix

**Humour décalé** : Les textes et slogans doivent refléter l'autodérision et le délire entre amis.

**Exemples de slogans déclinés** :
- Toujours + Con
- Toujours + En retard
- Toujours + Fauché
- Toujours + Bavard
- Toujours + Soif
- Toujours + Chaud
- Toujours + En terrasse
- Toujours + Connecté
- Toujours + Fatigué

## Spacing & Layout

- **Padding standard** : 16px, 24px, 32px
- **Margin standard** : 24px, 32px, 48px
- **Border radius** : 4px (petits éléments), 8px (cartes), 12px (sections)
- **Largeur max du contenu** : 1200px

## Accessibilité

- **Contraste minimum** : WCAG AA (4.5:1 pour le texte)
- **Textes alternatifs** : Tous les éléments visuels doivent avoir des descriptions
- **Focus states** : Tous les éléments interactifs doivent avoir des états focus visibles
- **Responsive** : Design mobile-first, testable sur tous les appareils

## Utilisation des Logos

### À faire ✓
- Utiliser le logo avec suffisant espace blanc autour
- Maintenir les proportions du logo
- Utiliser les couleurs officielles
- Appliquer le logo sur des fonds neutres

### À éviter ✗
- Modifier les couleurs du logo
- Étirer ou compresser le logo
- Placer le logo sur des fonds complexes
- Réduire le logo en dessous de 100px de largeur
